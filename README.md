
## compiling from source 

```
git clone https://github.com/josithg/go_checksum.git
```

```
cd go_checksum 
```

```
go mod tidy
```

```
go build -o main 
```
- or alternatively 
```
go build -o main.exe 
```

# Alternatively compiling from source in Linux 
- if this is too much work just run the build.sh if you are on Linux
```
git clone https://github.com/josithg/go_checksum.git
```

```
cd go_checksum
```

```
chmod +x ./build.sh
```

```
./build.sh
```
